# OSINT Investigation Report – High-Risk Online Investment Platform

This repository contains an open-source intelligence (OSINT)–based investigation conducted for educational and research purposes.

The objective of this project was to analyze and verify the legitimacy of an online investment platform using publicly available information and non-intrusive analysis techniques.

---

## Investigation Scope

The investigation focused on:

- Android application analysis
- Domain and infrastructure intelligence
- Certification and document verification
- Regulatory record verification
- Identity and transparency assessment

All analysis was performed using open-source tools and passive methods only.

---

## Tools Used

- VirusTotal  
- Mobile Security Framework (MobSF)  
- URLScan.io  
- Shodan  
- EXIFTool  
- SEC EDGAR Database  
- ASIC Company Register  

---

## Repository Structure

```report/ → Final investigation report (PDF/DOCX)
evidence/ → Screenshots and supporting materials
diagrams/ → Investigation flowcharts (PNG + HTML)

report/ → Report avaiable in both DOCX and PDF
```
---

## Important Notice

- All findings are based solely on publicly available information.
- No systems were accessed without authorization.
- No interaction with the investigated platform was performed.
- Indicators are sanitized to prevent misuse.

This project is intended strictly for **educational, research, and awareness purposes**.
